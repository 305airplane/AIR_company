#用如下公式计算离港延误时间：
#Departure delay = (ATD-STD)*24*60 (min)
#用如下公式计算进港延误时间：
#Arrival delay = (ATA-STA)*24*60 (min)
#离港延迟时间在25min内视为正常起飞，进港延迟时间大于或者等于15min视为航班延误。
